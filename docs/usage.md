# Usage Guide

## Installation

To install `django-tomselect`, run the following command:

```bash
pip install django-tomselect
```

## Configuration

### 1. Add to `INSTALLED_APPS`

Add `django_tomselect` to the `INSTALLED_APPS` setting in your Django project's `settings.py` file:

```python
INSTALLED_APPS = [
    # ...
    'django_tomselect',
    # ...
]
```
