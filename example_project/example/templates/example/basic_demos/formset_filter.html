Formset with filter_by Demo

Key Feature: Dependent Fields in Formsets

This demo shows how to use filter_by in Django's formsets. Each row has its own magazine/edition pair. When you select a magazine in any row, the edition dropdown in that row updates to show only editions from that magazine. Changes in one row won't affect other rows.

Notes:
  • The edition field uses filter_by=("magazine", "magazine_id")
  • Form prefixes (e.g., edition_filter-0-magazine) are automatically handled
  • Each row's edition field watches only its corresponding magazine field
  • Dynamically added rows also work correctly with the filter

Code Example

class EditionWithFilterFormsetForm(forms.Form):
    """Form demonstrating filter_by in formsets."""

    magazine = TomSelectModelChoiceField(
        config=TomSelectConfig(
            url="autocomplete-magazine",
            value_field="id",
            label_field="name",
            # ... other options
        ),
    )

    edition = TomSelectModelChoiceField(
        config=TomSelectConfig(
            url="autocomplete-edition",
            value_field="id",
            label_field="name",
            filter_by=("magazine", "magazine_id"),  # Key feature!
            # ... other options
        ),
        required=False,
    )

# Create the formset
EditionWithFilterFormset = formset_factory(
    EditionWithFilterFormsetForm, extra=2, can_delete=True
)
