Multiple Filter By Demo

This example demonstrates filtering articles by multiple field-based conditions combined with AND logic:

  • Select a Magazine to filter articles by that magazine
  • Select a Status to filter articles by that status
  • The Articles field will only show articles that match BOTH conditions
  • If you don't select a filter, that condition is ignored (shows all values for that field)
Code Example
from django_tomselect.app_settings import TomSelectConfig

# Multiple field filters - filter by magazine AND status
articles = TomSelectModelMultipleChoiceField(
    config=TomSelectConfig(
        url="autocomplete-article",
        filter_by=[
            ("magazine", "magazine_id"),  # Filter by selected magazine
            ("status", "status"),         # AND by selected status
        ],
    ),
)

How It Works

When you select both a magazine and a status, the autocomplete URL will include multiple f parameters: ?f='magazine__magazine_id=X'&f='status__status=Y'. The server applies these filters with AND logic, returning only articles that match both conditions.
